Detailed Description

XPm_Notifier - Notifier structure registered with a callback by app.

Field Documentation

void(*const XPm_Ntfier::callback)(struct XPm_Ntfier *const notifier)

Custom callback handler to be called when the notification is received.

The custom handler would execute from interrupt context, it shall return quickly and must not block! (enables event-driven notifications)
