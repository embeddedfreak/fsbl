          int XLoader_LoadAndStartSubSystemPdi 
          (
          XilPdi * 
          PdiPtr)
          
        

      





This function is used to load and start the PDI image. 


It reads meta header, loads the images as present in the PDI and starts based on hand-off information present in PDI


Parameters

  

    
PdiPtrPdi instance pointer

  

  




Returns
returns XLOADER_SUCCESS on success 


From the meta header present in PDI pointer, read the subsystem image details and load, start all the images


For every image,


    

  1. Read the CDO file if present
    2. 

  2. Send the CDO file to cdo parser which directs CDO commands to Xilpm, and other components
    3. 

  3. Load partitions to respective memories
    4. 



Check for Cfi errors


Set the Secondary Boot Mode settings to enable the read from the secondary device


Mark PDI loading is completed
