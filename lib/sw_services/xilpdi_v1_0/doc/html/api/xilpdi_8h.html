Below is the bit mapping of fields in the ATF Handoff parameters with that of Prtn header. 


The number of bits shifted is is based on the difference between these two offsets 


              ATFHandoffParams      PrtnHdr         Shift
Parameter     PrtnFlags     PrtnAttrb   difference

 


 Exec State 0 3 3 right ENDIAN 1 18 17 right SECURE 2 0 2 left EL 3:4 1:2 2 left CPU_A72 5:6 8:10

This function is used to validate the word checksum for the image header table and partition headers. 


Checksum is based on the below formulae Checksum = ~(X1 + X2 + X3 + .... + Xn)


Parameters

  

    
Bufferpointer for the data words.

    
Lenof the buffer for which checksum should be calculated. last word is taken as checksum for the data to be compared against

  

  




Returns
    

  • XST_SUCCESS for successful checksum validation
    • 

  • XST_FAILURE if checksum validation fails
    • 





Note


Checksum = ~(X1 + X2 + X3 + .... + Xn) Calculate the checksum
