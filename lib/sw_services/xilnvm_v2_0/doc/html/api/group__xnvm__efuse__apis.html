u32 XNvm_EfuseCheckAesKeyCrc ( u32  Crc)

This function performs the CRC check of AES key.

Parameters
CrcValueA 32 bit CRC value of an expected AES key.
Returns
  • XST_SUCCESS - On successful CRC check.
  • XNVM_EFUSE_ERR_CRC_VERIFICATION - On failure.
Note
For Calculating the CRC of the AES key use the XNvm_AesCrcCalc() function

u32 XNvm_EfuseWrite ( XNvm_EfuseData *  WriteNvm)

This function is used as a wrapper to program below eFuses AES key User key 0 User key 1 PPK0/PPK1/PPK2 hash IVs Revocation Ids User Fuses Secure and Control bits.

Parameters
WriteNvmPointer to the XNvm_EfuseData.
Returns
  • XST_SUCCESS - On Specified data write
    • Error code - On failure
Note
After eFUSE programming is complete, the cache is automatically reloaded so all programmed eFUSE bits can be directly read from cache.

References XNVM_EFUSE_ERR_BEFORE_PROGRAMMING, XNVM_EFUSE_ERR_INVALID_PARAM, and XNVM_EFUSE_ERR_NTHG_TO_BE_PROGRAMMED.

Referenced by XNvm_EfuseRevokePpk(), XNvm_EfuseWriteIVs(), and XNvm_EfuseWriteRevocationId().

u32 XNvm_EfuseWriteRevocationId ( u32  RevokeId)

This function writes Revocation eFuses.

Parameters
RevokeIdRevokeId number to program Revocation Id eFuses.

Example: If the revoke id to program is 64, it will program the 0th bit of the REVOCATION_ID_2 eFuse row

Returns
  • XST_SUCCESS - On successful write to eFuse.
  • Errorcode - On failure.

References XNVM_EFUSE_ERR_INVALID_PARAM, and XNvm_EfuseWrite().
