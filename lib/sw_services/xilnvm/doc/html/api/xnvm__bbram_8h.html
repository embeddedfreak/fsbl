Validates CRC of the key stored in BBRAM with CRC of input key. 


Parameters

  

    
Key- Pointer to key which is to be matched with key stored in BBRAM 

    
KeyLen- XNVM_128_BITS_AES_KEY_LEN_IN_BYTES for 128-bit AES key XNVM_256_BITS_AES_KEY_LEN_IN_BYTES for 256-bit AES key

  

  




Returns
XST_SUCCESS - Key is written to BBRAM XNVM_BBRAM_ERROR_PGM_MODE_ENABLE_TIMEOUT - Timeout during enabling programming mode XNVM_BBRAM_ERROR_PGM_MODE_DISABLE_TIMEOUT - Timeout during disabling programming mode XNVM_BBRAM_ERROR_AES_CRC_DONE_TIMEOUT - CRC validation check timed out. XNVM_BBRAM_ERROR_AES_CRC_MISMATCH - CRC mismatch XST_FAILURE - Unexpected error
