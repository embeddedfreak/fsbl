unsigned char *  Qy,
unsigned char *  r,

This function verifies ECDSA signature(r,s) of the hash 'z' (truncated to the group order size) using the public key 'Q'.

Parameters
ZPointer to the expected hash of the signed data.
QxPointer to the x co-ordinate of ECC point.
QyPointer to the y co-ordinate of ECC point.
rPointer to the r component of signature pair (r,s)
sPointer to the s component of signature pair (r,s)
Returns
XST_SUCCESS if verification was successful.
