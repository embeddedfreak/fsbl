About Xilinx Device Drivers

Xilinx provides full-featured device drivers for its Processor IP cores. The device driver architecture is described in the Device Driver Programmer Guide .

 

Device Drivers Summary

  • CAN Controller
    The Controller Area Network (CAN) driver supports the CAN core.
  • DDR ECC
    The Dual Data Rate (DDR) Error Correction Control (ECC) driver.
  • Ethernet 10/100 MAC Lite
    The Ethernet 10/100 MAC Lite driver supports the 10/100 Ethernet Lite core.
  • General Purpose I/O
    The General Purpose I/O (GPIO) driver supports the PLB and OPB GPIO cores.
  • Gigabit Ethernet MAC
    The 1 Gigabit Ethernet MAC driver supports the PLB 1 Gigabit EMAC cores.
  • Hardware ICAP
    The Hardware ICAP driver supports the HWICAP core.
  • Inter-Integrated Circuit
    The Inter-Integrated Circuit (IIC) driver supports the IIC cores.
  • Interrupt Controller
    The Interrupt Controller driver supports the INTC cores.
  • LL FIFO
    The llfifo driver supports the Axi FIFO Streaming core.
  • PCI Express bridge
    The PCI Express bridge driver supports the PCI Express core.
  • Serial Peripheral Interface
    The Serial Peripheral Interface (SPI) driver supports the SPI cores.
  • System Monitor
    The System Monitor driver supports the System Monitor ADC core/ XADC.
  • Timer/Counter
    The Timer/Counter driver supports the Timer cores.
  • UART Lite
    The UART Lite driver supports the UartLite cores.
  • UART 16450/16550
    The UART 16450/16550 driver supports the UART 16450/16550 cores.
  • USB
    The USB driver supports the USB 2.0 Device core
  • Watchdog Timer/Timebase
    The Watchdog Timer/Timebase driver supports the Watchdog Timer/Timebase cores.

 

Building Block Components

  • CPU
    CPU components comprise I/O functions specific to a processor. These I/O functions are defined in xil_io.h. These functions are used by drivers and are not intended for external use.
  • DMA
    DMA components comprise functions used for Direct Memory Access (DMA). Simple DMA, scatter-gather DMA, and multi-channel DMA are supported.
