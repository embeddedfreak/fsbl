◆ DPC_FSyncReset

#define DPC_FSyncReset (   BaseAddress)    DPC_WriteReg(BaseAddress, DPC_CONTROL, DPC_RST_AUTORESET) \

#include <dpc.h>

This macro resets a DPC instance, but differs from DPC_Reset() in that it automatically synchronizes to the SOF of the core to prevent tearing.

On the next rising-edge of SOF following a call to DPC_AutoSyncReset(), all of the core's configuration registers and outputs will be reset, then the reset flag will be immediately released, allowing the core to immediately resume default operation.

Parameters
BaseAddressis the Xilinx EDK base address of the DPC core (from xparameters.h)
Returns
None.
Note
C-style signature: void DPC_AutoSyncReset(u32 BaseAddress);

The registers will be automatically updated on the next rising-edge of the VBlank_in signal on the core. It is up to the user to manually disable the register update after a sufficient amount if time.

This function only works when the DPC core is enabled.

◆ DPC_Reset

#define DPC_Reset (   BaseAddress)    DPC_WriteReg(BaseAddress, DPC_CONTROL, DPC_RST_RESET) \

#include <dpc.h>

This macro resets an DPC instance.

This reset effects the core immediately, and may cause image tearing.

This reset resets the DPC's configuration registers, and holds the core's outputs in their reset state until DPC_ClearReset() is called.

Parameters
BaseAddressis the Xilinx EDK base address of the DPC core (from xparameters.h)
Returns
None.
Note
C-style signature: void DPC_Reset(u32 BaseAddress);
